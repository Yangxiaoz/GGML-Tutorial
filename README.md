# GGML-Tutorial

GGML机器学习库学习指南与代码示例



## 一、How to Start

1. Download 

```bash
git clone https://github.com/Yangxiaoz/GGML-Tutorial.git
cd GGML-Tutorial
```

2. CPU Build

```bash
cmake -B build
cmake --build build  -j 4
```

3. Using CUDA

```bash
cmake -B build -DGGML_CUDA=ON
cmake --build build -j 12
```
